# 灰度

一般用法：将整个页面变成灰色

取值：0-1 之间，可以为百分比

filter: grayscale(100%);

# 模糊/毛玻璃效果

一般用法：将结果模糊掉

取值：0px 以上

filter: blur(1px);

# 视差滚动 （兼容性略差）

MDN (opens new window) 设置当前元素滚动的速度比其他速度有一定的差别

  1. 设置父元素 perspective
  2. 目标元素设置 translateZ
.container {
    perspective: 1px;
}
.view {
    transform: translateZ(-1px) scale(2);
}

# CSS 实现轮播（兼容性略差）

  1. 父容器设置 scroll-snap-type (opens new window)
  2. 子列表设置 scroll-snap-aligin (opens new window)
.container {
    scroll-snap-type: x mandatory;
}
.item-view {
    scroll-snap-align: center;
}
