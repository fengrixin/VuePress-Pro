平时囤的一些库

# 常用控件

# Editor

  • Tinymac (opens new window) 功能强大的富文本

  • vue-quill-editor (opens new window) Vue 官方推荐，图片上传支持 element-ui

  • wangEditor (opens new window) 一个轻量、易用的富文本编辑器

# Dialog

  • sweetalert2 (opens new window) 一个好看、自适应且自定义性强的弹出框（零依赖）

# Upload

  • filepond (opens new window) 一个小巧的文件上传库

# Download

  • FileSaver (opens new window) 文件下载插件

# Image

  • tui.image-editor (opens new window) 功能齐全的基于 canvas 的在线图片编辑

  • img-2 (opens new window) 一个提高图片加载性能和体验的库，懒加载使用 web worker 模糊预览

# 图表

  • ECharts (opens new window) 功能强大的图标库 - 百度出品

  • AntV (opens new window) 功能强大的图标库 - 阿里出品

  • chart.xkcd (opens new window) 手绘风格的图表库

  • sketchviz (opens new window) 手绘风格的流程图

# Scroll

  • body-scroll-lock (opens new window) 滚动条（解决滚动穿透问题）

# Vue

  • Awesome Vue (opens new window) 适合在 Vue 中使用的一些库

# 移动端

  • pulltorefresh (opens new window) 移动端的下拉刷新插件

  • photoswipe (opens new window) 图片预览（可上下页切换，适配移动端）

# docx 转 html

  • mammoth (opens new window)
  • demo (opens new window)
